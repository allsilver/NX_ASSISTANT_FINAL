# HANDOFF_REQUESTS.md — 상호 요청 채널

> 남의 소유 파일을 직접 고치지 못한다. 변경이 필요하면 여기에 적고, 소유자가 처리한다.
> 처리되면 `[완료]` 표시.

## 형식
```
### [요청자 → 소유자] 제목
- 파일/영역:
- 무엇이/왜 필요:
- 제안(선택):
- 상태: 대기 / 진행 / 완료
```

---

## 요청 목록

### [예시] [codex → claude] nx_extrude 에 angle 인자 추가 제안
- 파일/영역: 계약 `ToolCall` / 카탈로그
- 무엇이/왜 필요: 경사 돌출 명령 지원 위해 angle 인자 필요
- 제안: 카탈로그 `nx_extrude` 에 `angle`(deg) 추가
- 상태: 예시 (실제 요청 아님)
